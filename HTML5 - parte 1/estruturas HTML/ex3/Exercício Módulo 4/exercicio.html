O Que É?

Notepad++ é um editor de texto e de código fonte de código sob a licença GPL. Suporta várias linguagens de programação rodando sob o sistema Microsoft Windows.

O Notepad++ é distribuído como um Software livre. O projeto foi hospedado no SourceForge.net, onde foi baixado mais de 27 milhões de vezes e ganhou duas vezes o prêmio "SourceForge Community Award" por melhor ferramenta de desenvolvimento.
