A história do HTML

O básico do HTML ao HTML5

O que é o "HTML"?

HTML é uma abreviação da expressão inglesa "HyperText Markup Language", que significa Linguagem de Marcação de Hipertexto. É uma linguagem de marcação utilizada para produzir páginas da Web, documentos HTML que podem se interpretados por navegadores.

A criação do HTML

Em 1980, um físico britânico chamado Tim Berners-Lee, propôs um projeto baseado no conceito de hipertexto para facilitar o compatilhamento e a solução combinada com a internet pública ganhou a atenção mundial.

As primeiras versões do HTML foram definidas com regras flexíveis, o que ajudou pessoas sem familiaridade com a publicação na Web, através do tempo, a utilizaçãode ferramentas para a criação HTML aumentou, assim com a rigidez sintática. Apesar disso por questões de retrocompatibilidade, os navegadores ainda conseguem interpretar páginas web que estão de ser um código HTML válido.

Etiquetas (tags)

Todo documento HTML possui tags, palavras entre < e > essas tags são comandos de formatação da linguagem. Uma tag pode ou não possuir atributos e valores. Os atributos são utilizados para modificar o resultado padrão do elemento.

Edição de arquivos HTML

Os documentos em HTML são arquivos de texto simples que podem ser criados e editados em qualquer editor de textos comum, como o Bloco de Notas do Windows, ou o TextEdit, do Mac OS.

Para facilitar a produção de documentos, no mercado existem editores de HTML específicos, com recursos sofisticados, que facilitam a realização de tarefas repetitivas, inserção de objetos, elaboração de tabelas e outros recursos, basicamente dividem-se em dois tipos:

Editores de texto fonte
Inserem automaticamente as etiquetas, orientando a inserção de atributos e marcação, como o Notepad++
Editores WYSIWYG
Oferecem um ambiente de edição visual, com um "esboço" do resultado final, como o Dreamweaver

O Web Hypertext application Technology Working Group (WHATWG) iniciou o trabalho do novo padrão HTML em 2004, enquanto o World Wide Web Consortium (W3C) estava se concentrando no desenvolvimento do XHTML 2.0. Em 2009 decidiu que a equipe de desenvolvimento XHTML 2.0 deveria parar seus trabalhos, e assim, descontinuar o padrão. Desta forma o W3C e o WHATWG passaram a travalhar juntos no desenvolvimento do HTML5.

O projeto do HTML5 foi bem recebido pelos desenvolvedores Web até então, e tornou-se tema na mídia em abril de 2010 depois que o CEO da Apple Inc., Steve Jobs emitiu uma carta pública intitulada "Reflexões sobre o Adobe Flash", onde ele concluía que o desenvolvimento do HTML5 tornaria o Adobe Flash não mais necessário tanto para assistir vídeo ou mesmo exibir qualquer conteúdo web.

A popularidade do HTML5

De acorodo com um relatório divulgado em 30 de setembro de 2011, 34% dos melhores Web sites do mundo estavam usando HTML5. A adaptação foi liberada por mecanismos de busca de redes sociais. Em 20 de outubro de 2011, o Facebook anunciou o lançamento do Centro de Recursos HTML, dando aos desenvolvedores ferramentas para construir, testar e implementar aplicação para o Facebook.

Logo do HTML5

Em 18 de Janeiro de 2011, a W3C introduziu um logo para representar o uso do HTML5. Diferente de outros emblemas anteriormente disponibilizados pela W3C, este não implica uma validação ou uso correto da linguagem. Desde o dia 1° de abril, este logotipo é oficial.
Segundo a W3C, o logotipo "Representa o HTML5, a pedra para aplicações web modernas".
